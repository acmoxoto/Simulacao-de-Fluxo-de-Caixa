Dados (Excel/CSV) para Simulação de Fluxo de Caixa

Conteúdo
• Transacoes: base de lançamentos (histórico + previsto) com Data_Caixa_Base e Valor_Sinal.
• Resumo_Mensal: consolidação mensal (Base) + colunas de cenário (Sim) por multiplicadores.
• Assumptions: parâmetros em azul para cenário simples (multiplicar entradas/saídas).
• Categorias: dicionário de categorias/subcategorias.

Dica (simulação mais avançada): use a aba/CSV 'Transacoes' para aplicar distribuições a Valor e Prazo_Dias e recalcular Data_Caixa.
